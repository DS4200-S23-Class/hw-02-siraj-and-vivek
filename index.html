Data Types

Airports

  • Airport dataset type is geometry
  • Row == item
  • airport_name == attribute
  • airport_name attribute is categorical
  • latitude == position
  • longitude == position

Google-Web

  • google-web dataset type is network
  • Row == item
  • FromeNodeId == link
  • ToNodeId == link

UFO

  • UFO dataset type is Table
  • Row == item
  • datetime == attribute
  • datetime attribute is ordinal
  • city == attribute
  • city attribute is categorical
  • state == attribute
  • state attribute is categorical
  • shape == attribute
  • shape attribute is categorical
  • duration == attribute
  • state attribute is quantitative
